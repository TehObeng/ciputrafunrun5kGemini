# Ciputra Batam Fun Run 5K

Landing page for the Ciputra Batam Fun Run 5K event site.

## Run Locally

Prerequisite: Node.js

```bash
npm install
npm run dev
```

## Production

```bash
npm install
npm test
npm run lint
npm run build
HOST=127.0.0.1 PORT=3200 npm run start
```
